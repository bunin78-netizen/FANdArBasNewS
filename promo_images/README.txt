FUNDARBAS Terminal Screenshots
================================

Place screenshots of the FUNDARBAS trading terminal in this folder.
Supported formats: .jpg, .jpeg, .png, .webp

The bot will randomly pick one image for each promo post and automatically
add the FUNDARBAS branded footer with a gold bar.

Tips:
- Use screenshots showing the trading interface, charts, portfolio view, etc.
- Recommended width: 800px or wider (will be auto-resized)
- The height will be capped at 600px, so landscape screenshots work best

If this folder is empty, the bot will generate a text-only promo card instead.
